# Learning Management System (LMS)

A comprehensive Learning Management System built with the MERN stack.

## TECH STACK

**Frontend**: React.js
**UI Styling**: HTML5, CSS3
**API**: REST API
**Backend**: Node.js + Express.js (Framework)
**Database**: MongoDB

## ARCHITECTURE

**Architecture Type**: Client – Server Architecture (3 Tier)
**Flow**: User (Browser) → React Frontend → REST API → Node.js + Express.js → MongoDB

## Installation

### 1. Clone the Repository
```bash
git clone https://github.com/srimathi-03/Online-Learning-Resource-Effectiveness-Analyzer
cd Online-Learning-Resource-Effectiveness-Analyzer
```

### 2. Install Dependencies
```bash
# Server
cd server
npm install

# Client
cd ../client
npm install
```

### 3. Run Application
```bash
# Terminal 1 (Server)
cd server
npm run dev

# Terminal 2 (Client)
cd client
npm start
```

## Features
- **Learner Dashboard**: Track progress, view enrolled courses.
- **Course Management**: Admin interface to create courses and add materials.
- **Assessment Engine**: Pre and Post tests with detailed analytics.
- **Analytics**: Admin overview of learner performance.
